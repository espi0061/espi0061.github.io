Get Started!

In order to start using and enjoying all the benefits that Bootstrap can provide, first it is necessary to link Bootstrap to the head of the HTML5 document being customized.

The best way to accomplish this is using the starter template that is provided at getbootstrap.com:

<!doctype html>
<html lang="en">
<head>
<meta charset="utf-8">
<meta name="viewport" content="width=device-width, initial-scale=1">

<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.0.2/dist/css/
bootstrap.min.css" rel="stylesheet" integrity="sha384-EVSTQN3/azprG1Anm3QDgpJLIm9Nao0Yz1ztcQTwFspd3yD65
VohhpuuCOmLASjC" crossorigin="anonymous">

<title>Hello, world!</title>
</head>

<body>
<h1>Hello, world!</h1>

<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.0.2/dist/js/
bootstrap.bundle.min.js" integrity="sha384-MrcW6ZMFYlzcLA8Nl+NtUVF0sA7MsXsP1UyJoMp4YLEuNSfAP+
JcXn/tWtIaxVXM" crossorigin="anonymous"></script>

</body>
</html>

Bootstrap System

What Is Reboot.css?

Reboot is a set of changes for specific CSS elements in a single file that initiates Bootstrap with a basic, consistent, and simple baseline to start building upon. It has a similar behavior than a normalize set of code. If a CSS file wants to be implemented on a project that also works with Bootstrap, then the link to the CSS style sheet must be placed after the link to Bootstrap's CDN. Otherwise, reboot.css would overwrite all of the changes made on CSS.

How Does Bootstrap Work?

Bootstap's system is simple and easy to learn. Once Bootstrap's CDN is linked to the head of the HTML5 document, one can start making changes to the design and layout of a web site without having to create and use CSS at all. Bootstrap already developed and prebuilt all the CSS code that is necessary to accomplish common design and layout patterns. In order to perform these changes, it is required to add specific classes to the html elements that want to be customized. From padding and margins, to colors and font sizes, all of this can be accomplish if the correct bootstrap class is applied.

Is Bootstrap Free to the Public?

Bootstrap is an open source framework which mean everyone on the internet can have access to it and can use it without extra charges or ownership issues. So, Bootstrap is a great resource for every web developer who desires to create layouts in a fast and efficient way!

Bootstrap Grid System

Bootstrap uses a unique grid system to allow developers create and customize different layouts within their web pages. This grid is mobile-first, uses flexbox terminology and functionality, counts with twelve columns to allocate better child elements, and has six different responsive tiers.

To make Bootstrap grid system work properly, there are some components that must be used:

  1. A parent element that wraps the whole grid with a class of .container
  2. An element that works as the row of the grid and contains all the child elements involved in the grid. This element would have a class of .row
  3. The child elements who are going to be allocated throughout the grid depending on layout customizations. These child elements must have a class of .col-# that specifies the amount of columns they are going to be placed at, .offset-# to move the elements certain number of columns, and .order-# to change the order in which they appear within the grid

Components

Bootstrap has developed a series of commonly used components within web sites that are ready to use and add to one's web design. Some of the components that Bootstrap has created include navs, accordions, alerts, buttons, and carousels.

Buttons

Bootstrap has originated a basic style for buttons and also some further customizations. When adding the .btn class to a button, it acquires a default padding and size that makes it more beautiful. However, if the developer desires to customize it even further, they can apply classes like .btn-primary, .btn-secondary, and .btn-info to add a desired color. Another option is to add classes like .btn-outline-primary, .btn-outline-secondary, and .btn-outline-info to add those same colors, but only to the button's outline.

Alerts!

By using .alert class Bootstrap adds basic styling like padding to the text that wants to be customized as alert. .alert-primary, .alert-secondary, .alert-info, and other alike classes add further customization like colors.
